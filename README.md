## Installation

FFastlib is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "FFastlib"
```

```
#import <FFastlib/FFastlib.h>
or
@import FFastlib;
```
